# Мой гараж

Статический сайт для ведения истории обслуживания автомобилей.

## Публикация на GitHub Pages

1. Загрузите содержимое этой папки в репозиторий GitHub.
2. В репозитории откройте `Settings -> Pages`.
3. Source: `Deploy from a branch`.
4. Branch: `main`, folder: `/root`.
5. Откройте выданную ссылку GitHub Pages.

## Общая синхронизация

Для общей семейной истории используйте Google Таблицу и Apps Script. Инструкция: `GOOGLE_SHEETS_SETUP.md`.

Без подключения Google Таблицы сайт тоже работает, но данные остаются только в браузере устройства.

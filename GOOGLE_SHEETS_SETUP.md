# Синхронизация через Google Таблицу

Схема такая:

`GitHub Pages -> сайт -> Google Apps Script -> Google Таблица`

GitHub хранит только файлы сайта. Таблица хранит общие данные по автомобилям и работам.

## 1. Создать таблицу

1. Создайте Google Таблицу.
2. Скопируйте ID из адреса:

```text
https://docs.google.com/spreadsheets/d/ЭТО_ID_ТАБЛИЦЫ/edit
```

## 2. Создать Apps Script

1. В таблице откройте `Расширения -> Apps Script`.
2. Удалите пустой код.
3. Вставьте содержимое файла `google-apps-script.gs`.
4. Вверху файла замените:

```js
const SPREADSHEET_ID = "PASTE_SPREADSHEET_ID_HERE";
const SYNC_PIN = "2115";
```

`SPREADSHEET_ID` — ID вашей таблицы.  
`SYNC_PIN` — простой общий PIN, который вы введете на телефонах.

## 3. Опубликовать веб-приложение

1. Нажмите `Развернуть -> Новое развертывание`.
2. Тип: `Веб-приложение`.
3. Выполнять как: `Я`.
4. У кого есть доступ: `Все`.
5. Нажмите `Развернуть`.
6. Скопируйте URL, который заканчивается на `/exec`.

## 4. Подключить на сайте

1. Откройте сайт.
2. В левом нижнем блоке нажмите `Локальный режим`.
3. Вставьте URL Apps Script.
4. Введите PIN.
5. Нажмите `Сохранить`.

После этого изменения будут отправляться в Google Таблицу. На другом телефоне введите тот же URL и PIN, затем нажмите `Загрузить из таблицы`.

## Важно

- Фото-графика хранится прямо в таблице как base64-текст, поэтому лучше использовать легкие изображения.
- Запись в таблицу идет через Apps Script. Это не настоящая авторизация как в Supabase, но для семейного закрытого использования с PIN обычно достаточно.
- Если поменяли код Apps Script, нужно заново нажать `Развернуть -> Управление развертываниями -> Изменить -> Новая версия`.
